# File Organizer (Python)

A simple desktop tool to automatically organize files into folders.

## Features
- Organizes Images, PDFs, Videos, Docs, Music
- Creates folders automatically
- Easy to use

## How to Use
1. Run the .exe file
2. Select the folder
3. Files will be organized automatically

## Technologies Used
- Python
- Tkinter (GUI)
